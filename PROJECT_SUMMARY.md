## 🚀 如何运行

### 方法1：使用启动脚本（Windows）
```bash
双击 start-server.bat
```

### 方法2：使用Python
```bash
cd nutrition-meal-planner
python -m http.server 8000
```

### 方法3：使用Node.js
```bash
cd nutrition-meal-planner
npx serve
```

然后访问 `http://localhost:8000`
