# 图片说明

由于这是MVP版本，暂时使用占位图片。

## 如何添加真实图片

1. 将菜品图片放入 `images/recipes/` 目录
2. 图片命名建议与菜谱ID对应，例如：
   - b001.jpg (番茄炒蛋)
   - l001.jpg (宫保鸡丁)
   - d001.jpg (清蒸鲈鱼)

3. 更新对应的JSON文件中的 `image` 字段：
   ```json
   "image": "./images/recipes/b001.jpg"
   ```

## 图片要求

- 格式：JPG 或 WebP（推荐WebP，体积更小）
- 尺寸：建议 800x600 像素
- 大小：压缩至 50KB 以下
- 内容：菜品成品图，光线充足，色彩鲜艳

## 占位图片

当前所有菜谱使用 `placeholder.jpg` 作为占位图。
你可以：
1. 使用在线占位图服务（如 placeholder.com）
2. 自己拍摄菜品照片
3. 从授权图库购买/下载
